## 2. 数据获取阶段

### 2.1 标的类型识别

**代码位置**: `src/search_service.py:1003-1021`

```python
_A_ETF_PREFIXES = ('51', '52', '56', '58', '15', '16', '18')

def is_index_or_etf(name: str, code: str = "") -> bool:
    """判断是否为指数或ETF"""
    # A股ETF: 6位代码，特定前缀
    if code and len(code) == 6 and code.startswith(_A_ETF_PREFIXES):
        return True

    # 国际ETF: 名称关键词
    _ETF_NAME_KEYWORDS = ('ETF', 'FUND', 'TRUST', 'INDEX', 'TRACKER', 'UNIT')
    if name and any(kw in name.upper() for kw in _ETF_NAME_KEYWORDS):
        return True

    return False
```

### 2.2 数据源优先级

| 优先级 | 数据源 | 说明 |
|--------|--------|------|
| -1 | Tushare | 配置Token后自动提升 |
| 0 | EFinance | 默认最高优先级 |
| 1 | AkShare | 综合数据源 |
| 2 | PyTDX | 通达信本地数据 |
| 3 | Baostock | 免费数据源 |
| 4 | YFinance | 美股专用 |

### 2.3 数据获取差异

| 项目 | 股票 | ETF |
|------|------|-----|
| 历史数据接口 | `ef.stock.get_quote_history()` | `ef.fund.get_quote_history()` |
| 实时行情接口 | `ef.stock.get_realtime_quotes()` | `ef.stock.get_realtime_quotes(['ETF'])` |
| 数据类型 | OHLCV | 净值数据 |
| 筹码分布 | 获取并分析 | 不获取 |

```python
class StockTrendAnalyzer:
    """股票趋势分析器"""

    def analyze(self, df: pd.DataFrame) -> Dict[str, Any]:
        """技术分析主入口"""
        return {
            # 均线系统
            'ma5': df['close'].rolling(5).mean(),
            'ma10': df['close'].rolling(10).mean(),
            'ma20': df['close'].rolling(20).mean(),
            'ma60': df['close'].rolling(60).mean(),

            # MACD指标
            'macd': self._calculate_macd(df),

            # RSI指标
            'rsi': self._calculate_rsi(df),

            # 量能分析
            'volume_ratio': self._calculate_volume_ratio(df),

            # 乖离率
            'bias': self._calculate_bias(df),

            # 技术形态
            'trend': self._detect_trend(df),
        }
```

### 5.2 AI 分析调用流程

**代码位置**: `src/analyzer.py:800-1100`

```python
async def analyze_stock(code: str, name: str, is_index_etf: bool = False):
    """AI 分析主入口"""

    # 1. 构建基础上下文
    context = build_analysis_context(code, name)

    # 2. 添加 ETF 特殊约束
    if is_index_etf:
        context += ETF_CONSTRAINT_WARNING

    # 3. 调用 LLM
    response = await litellm.acompletion(
        model=config.litellm_model,
        messages=[{"role": "user", "content": context}],
        fallback_models=config.litellm_fallback_models
    )

    # 4. 解析返回结果
    result = parse_analysis_response(response)

    return result
```

|------|--------|------|-----|

| | 业绩预期 | 公司财报 | 成分股表现 |

### 7.2 ETF 特殊约束详解

**代码位置**: `src/analyzer.py:948-956`

```markdown
⚠️ 指数/ETF 分析约束：

1. 风险分析范围：
   ✅ 关注: 指数走势、跟踪误差、市场流动性
```
